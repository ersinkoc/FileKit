# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added

- Initial release
- Core uploader with progress tracking and events
- Drop zone for drag and drop file uploads
- File validation (size, type, custom validators)
- Chunked upload support for large files
- Image preview generation and compression
- Upload queue with concurrency control
- React hooks: `useUploader`, `useDropZone`, `useFileSelect`, `useImagePreview`
- React components: `DropZone`, `FileInput`, `FileList`, `UploadProgress`, `ImagePreview`
- Full TypeScript support with strict mode
- Zero runtime dependencies
- Comprehensive documentation website
